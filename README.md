# Flutter E-Commerce App

A modern, high-performance e-commerce mobile application built with Flutter. This project demonstrates best practices in UI/UX design, state management (Provider), and clean architecture.

![App Logo](assets/images/logo.png)

## Features

-   **Modern UI/UX**: Compliant with Material 3 design guidelines.
-   **Dark Mode Support**: Seamless system-aware dark/light theme switching.
-   **Smooth Animations**: Powered by `flutter_animate` for a delightful user experience.
-   **Staggered Grid Layout**: Pinterest-style masonry layout for product discovery.
-   **Loading Shimmers**: Premium skeleton loading states instead of basic spinners.
-   **Parallax Product Details**: Immersive product viewing experience.
-   **Interactive Cart**: Slide-to-dismiss items and animated summaries.

## Tech Stack

-   **Framework**: Flutter & Dart
-   **State Management**: Provider
-   **Styling**: Google Fonts (Outfit), Custom Theme Extensions
-   **Animations**: flutter_animate
-   **Network Image**: cached_network_image

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/mel-adna/e-commerce-app.git
    ```

2.  **Install dependencies**
    ```bash
    flutter pub get
    ```

3.  **Run the app**
    ```bash
    flutter run
    ```
